# وكيل ذكاء اصطناعي متطور ذاتياً باستخدام Llama

هذا المشروع يوضح كيفية بناء وكيل ذكاء اصطناعي بسيط يتعلم ذاتياً باستخدام مكتبة `llama-cpp-python`.

## المتطلبات الأساسية

*   Python 3.8 أو أحدث
*   نموذج Llama GGUF (على سبيل المثال، من Hugging Face)

## إعداد المشروع

1.  **تنزيل نموذج Llama GGUF:**
    يجب عليك تنزيل نموذج Llama بتنسيق GGUF. يمكنك العثور على العديد من النماذج على [Hugging Face](https://huggingface.co/models?search=GGUF).
    
    **مثال:** ابحث عن نماذج مثل `Nous-Hermes-2-Mistral-7B-DPO.Q4_K_M.gguf` أو أي نموذج آخر يناسب احتياجاتك.
    
    بعد التنزيل، ضع ملف النموذج في نفس دليل المشروع أو قم بتحديث المسار في الكود.
    
    يمكنك أيضًا تعيين متغير البيئة `LLAMA_MODEL_PATH` ليشير إلى مسار النموذج الخاص بك.
    
    ```bash
    export LLAMA_MODEL_PATH="/path/to/your/model.gguf"
    ```

2.  **تثبيت التبعيات:**
    ```bash
    pip install llama-cpp-python
    ```

## تشغيل الوكيل

بعد تنزيل النموذج وتثبيت التبعيات، يمكنك تشغيل الوكيل باستخدام الأمر التالي:

```bash
python agent.py
```

سيقوم الوكيل بتنفيذ المهام المحددة في الكود وتحديث ذاكرته في ملف `agent_memory.txt`.

## هيكل المشروع

*   `agent.py`: الكود الرئيسي لوكيل الذكاء الاصطناعي.
*   `agent_memory.txt`: ملف لتخزين ذاكرة الوكيل (يتم إنشاؤه تلقائياً).
*   `README.md`: هذا الملف.
